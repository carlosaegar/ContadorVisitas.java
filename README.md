Ejercicio 16, archivo BC3 II, REALIZADO EN CLASE Enunciado; 3. Maps: Contador de visitas a páginas
Crea un programa que use un HashMap<String, Integer> para contar cuántas veces se visita cada página 
de un sitio web (por nombre de página).
El programa debe:
• Leer en un bucle nombres de página (por ejemplo: "inicio", "contacto", "productos").
• Cada vez que se introduce una página:
o Si no existe en el mapa, se añade con valor 1.
o Si ya existe, se incrementa su contador.
• Al escribir "fin" se deja de pedir páginas y se muestran todas con su número de visitas
